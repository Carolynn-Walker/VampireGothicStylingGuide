Element Stylings

Element Use Cases Styling Rules
Title Only use one, for SEO. Should reiterate the title of the page. Ideally matches the text displayed in the browser tab. Bolded. Special Font TBD
Headings Includes all heading tags except H1. Section headings used to announce/describe new parts of the site Typically left alone. Srisakdi font. Generally centered.
Subheadings Things like sub titles, additional information placed right below headings, etc. Think second, third, etc., tier in APA styling Italicized. Either centered with the heading it is expaning or left-aligned when there will be multiple subheadings in a section. Uses Srisakdi as well as the font.
General text All other text on the site. Anything that is not a heading. Uses Modern Antiqua font.
